Implement an inverse geocoding functionality that accurately retrieves and displays the building number from geographic coordinates. The retrieved building number must be automatically populated into the corresponding address field in the system. Ensure the implementation includes:

1. Precise coordinate-to-address conversion with building number extraction
2. Seamless integration with existing address fields
3. Validation to confirm the building number exists in the geocoded results
4. Error handling for cases where building number cannot be determined
5. Clear visual indication when the building number has been automatically applied

The solution should maintain data consistency between the geocoded results and address fields, with proper synchronization when either value is modified. Include unit tests to verify:
- Correct building number extraction from various coordinate formats
- Proper field population under different scenarios
- Handling of edge cases and invalid inputs
