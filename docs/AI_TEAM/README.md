# 🤖 AI Team Documentation

This folder contains the complete context, decisions, plans, and activity logs for the AI-assisted development of Matrix Delivery Platform.

## Purpose
- Onboard new AI agents or team members instantly
- Preserve full context across sessions (no knowledge loss)
- Track all decisions, plans, and actions

## Files
- `README.md` — This file
- `ONBOARDING.md` — Full project context for new agents/members
- `DECISIONS.md` — All architectural and business decisions
- `ACTIVITY_LOG.md` — Chronological log of all work done
- `SPRINT_PLAN.md` — Current sprint priorities
- `MONETIZATION.md` — Revenue strategy and opportunities
- `SALAMHACK_2026.md` — Hackathon preparation plan

## Key Principle
**Every significant discussion, decision, or action MUST be logged here.**
If an AI agent's quota runs out, the next agent reads these files and continues seamlessly.
