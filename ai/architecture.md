Matrix Delivery Platform

Stack
- Backend: Node.js + Express
- Frontend: React
- Database: PostgreSQL
- Maps: OpenStreetMap

Current capabilities
- Customers place delivery orders
- Drivers deliver orders

New feature
Vendor Marketplace

Capabilities required:
- Vendors register
- Vendors create stores
- Vendors add items
- Items belong to nested categories
- Vendors create special offers
- Customers browse nearby stores
- Customers add items to cart
- Orders include platform commission (10%)
